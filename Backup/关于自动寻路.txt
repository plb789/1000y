新增功能： 我刚刚在 MapEngine 中添加了一个公开的 navigateTo(targetX, targetY) 方法，方便任务系统调用：

```
// 任务系统调用示例
mapEngine.navigateTo(targetX, 
targetY);
```
方法说明：

参数 类型 说明 targetX number 目标格子X坐标 targetY number 目标格子Y坐标 返回值 boolean 成功返回 true，失败返回 false（目标不可达或无路径）

使用场景：

```
// 任务系统引导玩家前往NPC位置
if (mapEngine.navigateTo(npcX, 
npcY)) {
  console.log('开始自动寻路...');
} else {
  console.log('目标位置不可到达');
}
```
