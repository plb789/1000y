### 📊 各等级怪物详细分布

#### 🟢 **低级怪区域** (距离出生点 80-200 格)

**适合：1-2级新手**

| 怪物 | 等级 | 数量 | 分布范围 | 示例坐标 |
|------|------|------|----------|----------|
| 🐔 野鸡 | Lv1 | 5个 | 半径80-200 | (620, 480), (380, 550), (500, 680)... |
| 🐰 兔子 | Lv1 | 5个 | 半径80-200 | (580, 350), (420, 620), (650, 520)... |
| 🐭 老鼠 | Lv2 | 5个 | 半径80-200 | (350, 450), (600, 380), (480, 700)... |

**特点：**
- ✅ **最靠近出生点** - 新手走出安全区就能看到
- ✅ **血量低 (40-60HP)** - 容易击杀
- ✅ **被动型AI** - 不会主动追击（除非被攻击）
- ✅ **经验少 (8-15exp)** - 适合练手

**示例位置（相对出生点的偏移）：**
```
         ↑ 北 (Y轴减小)
         |
    (500,300) ← 野鸡可能在这里
         |
    (400,500) ← 兔子可能在这里 ← — — (500,500)出生点 → (600,500) → 老鼠可能在这里
         |
    (500,700) ← 更多低级怪
         |
         ↓ 南 (Y轴增加)
```

**示例位置：**

```

```

### 围绕玩家生成一圈怪物（用于压力测试）

```bash
# 在玩家周围360度生成怪物
for angle in 0 45 90 135 180 225 270 315; do
  # 计算坐标 (假设玩家在500,500，半径150)
  X=$(echo "500 + 150 * c($angle * 3.14159 / 180)" | bc -l | cut -d. -f1)
  Y=$(echo "500 + 150 * s($angle * 3.14159 / 180)" | bc -l | cut -d. -f1)
  
  curl -s -X POST http://localhost:8080/api/monster/gm/spawn \
    -H "Content-Type: application/json" \
    -d "{\"base_id\": 104, \"map_id\": 1, \"x\": $X, \"y\": \"$Y\", \"count\": 1}"
done
```

## 🎨 可视化工具

### Python脚本生成分布热力图

```python
import matplotlib.pyplot as plt
import numpy as np

# 地图配置
MAP_WIDTH = 1000
MAP_HEIGHT = 1000
CENTER_X, CENTER_Y = 500, 500

# 创建画布
fig, ax = plt.subplots(1, 1, figsize=(10, 10))

# 绘制地图边界
ax.set_xlim(0, MAP_WIDTH)
ax.set_ylim(0, MAP_HEIGHT)
ax.set_aspect('equal')

# 绘制区域
# 低级怪区域 (绿色)
theta = np.linspace(0, 2*np.pi, 100)
for r in [80, 200]:
    x = CENTER_X + r * np.cos(theta)
    y = CENTER_Y + r * np.sin(theta)
    ax.plot(x, y, 'g--', alpha=0.5, label=f'低级怪区域 ({r}格)' if r==80 else '')

# 中级怪区域 (黄色)
for r in [150, 300]:
    x = CENTER_X + r * np.cos(theta)
    y = CENTER_Y + r * np.sin(theta)
    ax.plot(x, y, 'y--', alpha=0.5, label=f'中级怪区域 ({r}格)' if r==150 else '')

# 高级怪区域 (红色)
for r in [250, 450]:
    x = CENTER_X + r * np.cos(theta)
    y = CENTER_Y + r * np.sin(theta)
    ax.plot(x, y, 'r--', alpha=0.5, label=f'高级怪区域 ({r}格)' if r==250 else '')

# 标记出生点
ax.plot(CENTER_X, CENTER_Y, 'b*', markersize=20, label='出生点')
ax.annotate('出生点\n(500,500)', (CENTER_X, CENTER_Y), 
            textcoords="offset points", xytext=(0,20), ha='center')

# 模拟生成一些怪物位置（示例）
np.random.seed(42)
for i in range(25):
    if i < 15:  # 低级怪
        angle = np.random.uniform(0, 2*np.pi)
        dist = np.random.uniform(80, 200)
        color = 'green'
        marker = 'o'
    elif i < 23:  # 中级怪
        angle = np.random.uniform(0, 2*np.pi)
        dist = np.random.uniform(150, 300)
        color = 'orange'
        marker = 's'
    else:  # 高级怪
        angle = np.random.uniform(0, 2*np.pi)
        dist = np.random.uniform(250, 450)
        color = 'red'
        marker = '^'
    
    x = CENTER_X + dist * np.cos(angle)
    y = CENTER_Y + dist * np.sin(angle)
    ax.plot(x, y, marker=marker, color=color, markersize=8, alpha=0.7)

ax.legend(loc='upper right')
ax.set_title('地图1（新手村）怪物分布示意图', fontsize=16, fontweight='bold')
ax.set_xlabel('X 坐标')
ax.set_ylabel('Y 坐标')
ax.grid(True, alpha=0.3)

plt.tight_layout()
plt.savefig('monster_distribution_map1.png', dpi=150, bbox_inches='tight')
plt.show()
print("✅ 分布图已保存为 monster_distribution_map1.png")
```

## 💡 设计理念总结

### 为什么采用环形分布？

1. **渐进式难度**
   - 新手不会被高级怪秒杀
   - 有明确的成长路径
   - 符合RPG游戏设计原则

2. **空间利用合理**
   - 避免所有怪物堆在一起
   - 探索感强（不同方向有不同怪）
   - 减少资源竞争

3. **性能优化**
   - 玩家只加载视野内的怪物
   - 远处的怪物不影响帧率
   - 支持大地图无缝衔接

4. **社交性**
   - 低级区域成为新手聚集地
   - 方便老玩家帮助新人
   - 形成自然的"安全区→冒险区"过渡
