## 功能概述

### 已实现的核心功能 ✅

#### 1. 攻击判定系统
- **距离检测** - 基于坐标的欧几里得距离计算，支持近战/远程攻击范围配置
- **命中判定** - 基于攻击方命中率 vs 防御方闪避率的概率计算（30%-95%）
- **格挡机制** - 防御力影响格挡率，成功格挡减少30%-50%伤害
- **冷却管理** - 普通攻击1.5秒冷却，技能独立冷却（支持全局冷却管理器）

#### 2. 伤害计算引擎
- **基础公式**: `伤害 = 攻击力 - 防御力 * 0.5 + 技能加成`
- **暴击系统** - 暴击率判定 + 暴击伤害倍率（默认150%）
- **闪避机制** - 完全免疫伤害（显示"闪避"飘字）
- **格挡减伤** - 减少部分伤害并显示"格挡"提示
- **伤害波动** - ±10%随机波动，增加战斗不确定性
- **技能系数** - 不同武学类型有不同伤害系数（外功150%、斧法160%等）

#### 3. 怪物AI系统（状态机驱动）
```
空闲(Idle) → 巡逻(Patrol) → 警戒(Alert) → 追击(Chase) → 战斗(Combat)
                                                              ↓
                                                         死亡(Dead) → 复活(Respawn) → 空闲(Idle)
                                                              ↓
                                                         逃跑(Flee) → 巡逻(Patrol)
```

**AI行为特性**:
- **被动型 (AIType=0)**: 不主动攻击，被攻击后追击
- **主动型 (AIType=1)**: 正常追击范围，低血量(<5%)逃跑
- **攻击型 (AIType=2)**: BOSS/精英，追击范围翻倍，永不逃跑

**巡逻系统**:
- 自动生成3-5个巡逻点（出生点周围3-6格）
- 沿路径循环移动
- 完成一圈后有30%概率休息

#### 4. 死亡掉落系统
- **经验掉落**: 根据等级差修正（±10级影响50%经验）
- **金币掉落**: 配置范围内随机值
- **物品掉落**: 基于万分比概率的掉落组Roll机制
- **复活机制**: 可配置复活时间（30-600秒）

## 服务端实现详解

### 1. 核心算法 ([battle.go](Server/GameService/Battle/battle.go))

#### CalculateFinalDamage() - 完整伤害计算
```go
func CalculateFinalDamage(attacker, defender *BaseFighter, skillBonus int) 
    (int, bool, bool, bool, float64) {
    
    // 1. 命中检测 (30%-95%)
    if !CalculateHit(attacker, defender) {
        return 0, false, true, false, 0  // 闪避
    }
    
    // 2. 格挡检测 (最高40%概率, 减少30-50%伤害)
    isBlocked, blockReduction := CalculateBlock(attacker, defender)
    
    // 3. 基础伤害 = 攻击 - 防御*0.5
    damage, isCrit := CalculateDamage(attacker, defender, skillBonus)
    
    // 4. 应用格挡减伤
    if isBlocked {
        damage = int(float64(damage) * (1 - blockReduction))
    }
    
    return damage, isCrit, false, isBlocked, blockReduction
}
```

#### CheckAttackRange() - 距离检测
```go
func CheckAttackRange(attackerX, attackerY, targetX, targetY int, attackRange int) 
    (bool, float64) {
    distance := math.Sqrt(dx*dx + dy*dy)
    return distance <= float64(attackRange), distance
}
```

### 3. 完整攻击流程 ([battle_service.go](Server/GameService/Battle/battle_service.go))

#### PlayerAttackMonster() - 玩家攻击怪物
```go
func (s *Service) PlayerAttackMonster(roleID uint64, monsterInstanceID uint64, skillID uint32) 
    *PlayerAttackResult {
    
    // 1. 获取玩家属性
    playerFighter, _ := s.getPlayerFighter(roleID)
    
    // 2. 获取怪物实例
    monster, _ := monsterSvc.GetMonster(monsterInstanceID)
    
    // 3. 检查攻击距离 (近战1格)
    inRange, _ := battle.CheckAttackRange(playerPos.X, playerPos.Y, 
                                           monster.X, monster.Y, 1)
    
    // 4. 检查攻击冷却 (1.5秒)
    GlobalCoolDownManager.IsCoolingDown(roleID, 0)
    
    // 5. 计算最终伤害
    damage, isCrit, isMiss, isBlocked, _ := 
        battle.CalculateFinalDamage(playerFighter, monsterFighter, skillBonus)
    
    // 6. 应用伤害
    newHP, isDead := monsterSvc.MonsterTakeDamage(monsterInstanceID, damage)
    
    // 7. 通知AI系统
    s.notifyMonsterHurted(monsterSvc, monsterInstanceID, roleID)
    
    // 8. 处理死亡掉落
    if isDead {
        exp, gold, drops, _ := monsterSvc.MonsterDie(monsterInstanceID)
        s.rewardPlayer(roleID, exp, gold)
    }
    
    return result
}
```

        case 'attack_result':

            if (data.target_id === this.player.id) {
                this.player.hp = data.current_hp;         // 更新自身血量

            break;

            break;

## 通信协议

### 前端 → 服务端（攻击请求）

**协议号**: `0x1001` (普通攻击)

```json
{
    "cmd": "attack",
    "target_id": 12345,       // 怪物实例ID
    "target_type": 2,         // 2=怪物
    "skill_id": 0,            // 0=普通攻击
    "x": 10,                  // 攻击者位置（可选）
    "y": 20
}
```

3. **点击怪物** → 选中（青色虚线框）→ 显示怪物信息
4. 如果在攻击范围内 → **自动发起攻击**
5. 如果距离过远 → **自动移动到怪物附近**
