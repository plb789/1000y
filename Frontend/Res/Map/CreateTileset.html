📋 使用说明

  • 步骤1: 拖拽或选择一张完整的大图（包含多个瓦片拼成的图）
  • 步骤2: 设置瓦片尺寸和图集列数
  • 步骤3: 点击"预览切片"确认切片是否正确
  • 步骤4: 编辑每个瓦片的属性（通行/阻挡/名称）
  • 步骤5: 点击"下载完整包"导出 PNG + JSON 元数据
  • 步骤6: 将导出的文件放入 Res/Map/ 目录
  • 步骤7: 在地图编辑器中加载图集（会自动读取JSON）

🏷️ 瓦片索引说明

  • 索引从 1 开始，与千年服务端地图格式一致
  • 通行属性：玩家可以走到该瓦片上
  • 建筑物：作为物体层渲染，会遮挡地面
  • 导出的 JSON 文件需要与 PNG 同名
